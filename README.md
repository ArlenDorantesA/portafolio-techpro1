### ÌNDICE



*******

El presente proyecto es creado para el bootcamp de **Tecnolochicas PRO Summer Camp** 💜.

El propósito del el sitio web es mostrar el portafolio de proyectos de un(a) desarrollador(a) y su experiencia 👩🏻‍💻.
